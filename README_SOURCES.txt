Used for Capacitive Sensors: https://github.com/admarschoonen/TouchLib 

Used for FSR, Servos, Random generation, LEDS: https://www.arduino.cc/en/Reference/

To run: Upload Permabuild.ino into Arduino Mega 2560. Make sure to add files TouchLib.h before uploading. If necessary files missing then 
Install entire touch lib library at specified github source above.
